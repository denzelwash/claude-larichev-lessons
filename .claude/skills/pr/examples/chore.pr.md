# Пример: chore PR (без чек-листа ревью)

**Заголовок:** `chore(deps): обновить vite до 5.4`

**Тело:**

```
## Что сделано
- Обновлён vite с 5.2.8 до 5.4.1 в `apps/frontend`
- Обновлены связанные пакеты: `@vitejs/plugin-vue`, `vite-tsconfig-paths`
- Проверена сборка `npm run build:web` — без ошибок
```

---

**Примечания:**
- Секция `## API` отсутствует — бэкенд не затронут.
- Секция `## Чек-лист для ревью` отсутствует — тип `chore` не требует.
- Для `docs`, `ci`, `build` — то же самое: только `## Что сделано`.
